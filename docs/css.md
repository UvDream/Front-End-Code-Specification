# CSS规范
-  ## 代码需要格式化后提交
- ## Css代码禁止大写
- ## 尽量少使用选择器
- ## 代码缩进在代码格式化文件
- ## 颜色值中不需要有空格,且取值不要带不必要的0
> 推荐
```
.net {
    color: rgba(255,255,255,.5);
}
```
> 不推荐
```
.net {
    color: rgba(255,255,255,0.5);
}
```
- ## 属性值十六进制数值能用简写尽量简写
- ## 不要为0指定单位
> 推荐:
```
.net {
    margin: 0 10px;
}
```
> 不推荐
```
.net {
    margin: 0px 10px;
}
```
- ## 属性值在引用的时候统一使用单引号
